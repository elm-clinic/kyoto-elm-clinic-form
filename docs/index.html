〜 エ ル ム に つ い て 〜

「切らずに、きれいに」
 エルムクリニックは、全国７院を展開する美容皮膚科です。 おかげさまで、今年をもって10周年となります。 地域とお客様の心に深く根ざして、笑顔と技術を追求する。 その初心を忘れず、京都院をリニューアルオープンします！


私たちのクリニックでは、
地域と患者さまの心に深く根ざして
成長していきたいという想いがあります。
その理念に基づき、
エルムの木をモチーフにしました。

〜 医師からのメッセージ 〜
「より美しく、より健康でいたい。」 誰もが願うことではないでしょうか？ 当院では、最先端の美容医療で、メスを使わずにあなたの自然な美しさを引き出します。 お悩み・ご要望を丁寧にお聞きし、あなたにぴったりの道をご案内しますので、 まずはお気軽にカウンセリングへお越しください。
 また、お肌の健康には、身体の健康が欠かせません。 京都院では、漢方診療・栄養療法も行っております。気になる方はぜひご相談ください。
 お肌も身体も、そして心も健康に。 スタッフ一同、皆さまのご来院を心よりお待ちしております。

〜 Q & A 〜


  • どうやって予約すればいいですか？

    以下のボタンからご予約を承っております！

    WEB からご 予約

  • どこにありますか？

     藤井大丸さんの1軒南側にある、1階が鏡屋さんのビルの4階です。
     住所は、京都府京都市下京区寺町通 四条下る貞安前之町594 です！

  • 予約なしで直接クリニックに行っても治療をしてもらえますか？

    エルムクリニックでは、スムーズなご来院、患者様の貴重なお時間を大切にさせていただくため、完全予約制となっております。ご理解いただきますよう、よろしくお願いいたします！

  • 予約後の返信がありません！

    ドメイン・指定受信・本文にURLがあるメールの受信拒否などの制限をかけている方は、エルムクリニック京都院からのメールを受信できない場合がございます。また、メールサーバの混雑、不具合等によりデータが損失してしまった可能性もございます。再度送信いただけますと幸いです！

  • カウンセリングと施術は、別々に予約が必要ですか？

    アートメイクを除いて、空きがあれば同日の施術が可能です。 施術のご予約がない場合はお待たせしてしまう可能性がありますので、予約時に同日施術をご希望との旨をお伝えください。また、カウンセリングの最終受付は18時となっております。

  • 駐車場はありますか？

    エルムクリニック専用の駐車場はございません。近隣の駐車場をご利用ください。

  • 診察日に持っていった方がいいものはありますか？

    医療ローンをお使いになる場合は、身分証明書が必要となります！

  • 未成年ですが、施術を受けることは可能でしょうか？

    19歳以上、もしくは高校をご卒業されていれば、同意書は不要となります。 また、高校生以下の方は、保護者の方の同伴・もしくは同意書をお持ちいただければ施術が可能です。下記リンクより親権者同意書をダウンロードいただき、直筆で記入いただいた上、来院時にご提出ください。また、18歳以下の方は保護者の方の同席が必要となります。
